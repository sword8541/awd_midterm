    General information

  • Get/POST/PUT/DELETE http://127.0.0.1:8000/api/vehicle/[VEHICLE ID] -return the vehicle and all we know about it, it uses django rest framework and data validated by serializer,you can also add,update and delete any entry
    • http://127.0.0.1:8000/api/vehicles/15856
  • POST http://127.0.0.1:8000/create_vehicle/ -add a new vehicle for renting (this links to a simple page with a form that we make the post request)
    • http://127.0.0.1:8000/create_vehicle/

    Vehicles Preference

  • Get http://127.0.0.1:8000/api/vehicles -return a list of vehicles with all the info we know
    • http://127.0.0.1:8000/api/vehicles
  • Get http://127.0.0.1:8000/api/vehicles/highest-ranking -return number of vehicles(with at least 1000 user reviews) and its average ranking(1-5) so that we have an overview of people’s car rental preference at that time
    • http://127.0.0.1:8000/api/vehicles/highest-ranking
  • Get http://127.0.0.1:8000/api/vehicles/fuel-type -return number of vehicles grouped by fuel type in descending order(ie ELECTRIC,HYBRID,GAAOLINE) so that we have an overview of people’s car rental preference at that time
    • http://127.0.0.1:8000/api/vehicles/fuel-type
  • Get http://127.0.0.1:8000/api/vehicles/vehicle-type -return number of vehicles grouped by vehicle type in descending order(ie car,suv,minivan) so that we have an overview of people’s car rental preference at that time
    • http://127.0.0.1:8000/api/vehicles/vehicle-type
  • Get http://127.0.0.1:8000/api/vehicles/vehicle-make -return number of vehicles grouped by vehicle makers in descending order(ie Toyata,Tesla) so that we have an overview of people’s car rental preference at that time
    • http://127.0.0.1:8000/api/vehicles/vehicle-make
  • Get http://127.0.0.1:8000/api/vehicles/ev-make -return number of pure Electric vehicles grouped by vehicle makers in descending orde so that we have an idea of which company made most EVs at that time
    • http://127.0.0.1:8000/api/vehicles/ev-make

    Vehicle Owners

  • Get http://127.0.0.1:8000/owners/top5 -return a simple page with top5 owners with most vehicles for renting in descending order
    • http://127.0.0.1:8000/owners/top5
  • Get http://127.0.0.1:8000/api/owners/top5 -api version of top5 owners with most vehicles for renting(guess how many vehicles they had) in descending order
    • http://127.0.0.1:8000/api/owners/top5

    Cities

  • Get http://127.0.0.1:8000/api/cities/ev-top5 -top5 cities with most EVs for renting(now we have an idea which city is green power pioneer ) in descending order
    • http://127.0.0.1:8000/api/cities/ev-top5
  • Get http://127.0.0.1:8000/api/cities/vehicle-by-fuel-type -return a detail list of vehicles grouped by fuel type in each city
    • http://127.0.0.1:8000/api/cities/vehicle-by-fuel-type
